# Human-3D-Viewer

Интерактинвная 3D-модель человека для web-
browser на основе цифровой 2D-фотографии с применением нейроной сети PIFuHD и библиотеки WEBGL для синтеза 2D/3D графики.
** Нейросеть PIFuHD:**
- https://shunsukesaito.github.io/PIFuHD/
